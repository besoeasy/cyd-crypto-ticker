Use a secure page in Chrome, Edge, or another Chromium browser with Web Serial enabled.

If "Failed to initialize" appears

  1. Unplug the board, then plug it back in with a data USB cable.
  2. Hold BOOT.
  3. While still holding BOOT, click the install button and select the serial port.
  4. If the board has a reset button, tap RESET or EN once while still holding BOOT.
  5. Release BOOT only after the install dialog starts preparing or flashing.

Before You Start

  1. Use a data USB cable, not a charge-only cable.
  2. Close serial monitors or other apps holding the ESP32 port.
  3. Some CYD boards do not auto-enter the bootloader, so manual BOOT plus RESET/EN may be required.

Board Variants

CYD is the default board profile. CYD2USB uses the alternate TFT inversion setting for boards with USB mounted on the opposite edge.

After Flashing

  1. Let the device reboot after install.
  2. Connect to the temporary Wi-Fi access point created by the ticker.
  3. Enter Wi-Fi credentials and your CoinGecko coin IDs.

Troubleshooting

  1. If the install dialog cannot see the port, reconnect the board and refresh the page before trying again.
  2. If initialization fails, repeat the manual boot sequence above and keep BOOT pressed longer.
  3. If flashing starts but fails, disconnect and retry with no serial terminal open.
  4. If the screen colors look inverted after install, switch to the other board variant and flash again.
